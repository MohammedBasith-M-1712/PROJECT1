# PROJECT1
Employee Performance Data Analysis Project
Project Overview

This project focuses on cleaning, analyzing, and visualizing employee performance data using Python.
The dataset contains employee details such as department, salary, experience, performance score, and location.

The main objective of this project is to perform data preprocessing and generate meaningful insights through visualizations.

Dataset Name

Employee_Performance_Dataset.csv

Technologies Used
Python
Pandas
Matplotlib
Features
Load and read CSV dataset
Check missing values
Remove duplicate records
Generate statistical summaries
Analyze department-wise salary
Visualize:
Average Salary by Department
Performance Score Distribution
Experience vs Salary

Step 1: Install Required Libraries
